**[CompSem -- NLI](https://github.com/dylanjgoode/CompSem)**

An NLP project that implements and trains multiple neural sentence encoders for natural language inference, using the SNLI dataset and SentEval for downstream evaluation.

### Highlights

- **Four encoders**: AWE, LSTM, BiLSTM, and BiLSTM with max pooling.
- **Training and evaluation scripts** for SNLI and SentEval.
- **Pretrained checkpoints** available for reuse.
